*** ImageLinks Lite WordPress Plugin Changelog ***

version 1.3.2 22/12/2016
 * Fix: Bug with hotspot preview images

version 1.3.1 29/11/2016
 * Feature: Image url can be local relative to the upload folder or full
 * Fix: Editor can't save a popover content in some cases

version 1.3.0 25/11/2016
 * Feature: Added WordPress editor for a popover content
 * Feature: Allow move hotspots via arrow keys
 * Feature: Now you can add your own theme to plugin folder
 * Improvement: Updated the builder interface

version 1.2.0 17/10/2016
 * Feature: Added a new parameter 'popoverLazyload'
 * Feature: Added a new parameter 'popoverShow'
 * Feature: Allow shortcodes within a popover content
 * Fix: When special characters are used in popovers the plugin doesn't work
 * Improvement: Updated the builder

version 1.1.0 18/07/2016
 * Feature: Added a new theme 'dots'
 * Feature: Added a new parameter 'hotSpotBelowPopover'
 * Feature: Added a new parameter 'content' for the hotspot definition
 * Feature: Added new location types (top-left, top-right, bottom-left, bottom-right)
 * Fix: Bug with z-order of popover windows
 * Improvement: Updated the builder

version 1.0.0 20/06/2016
 * Initial release
